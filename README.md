# Customer Behavior Analysis 📊

## Power BI Dashboard

![Power BI Dashboard](images/dashboard.png)

## Overview

This project analyzes customer shopping behavior data to discover useful business insights.  
The complete analytics workflow is performed using Python, SQL, and Power BI.

The project focuses on understanding customer patterns, purchase trends, and key factors affecting business performance.


## Dataset

The dataset contains customer shopping information including:

- Customer demographics
- Purchase details
- Product categories
- Transaction information
- Spending behavior


## Tools Used

- Python (Pandas, Matplotlib, Seaborn)
- SQL (PostgreSQL / MySQL / SQL Server)
- Power BI
- Excel


## Project Workflow

### 1. Data Cleaning & EDA (Python)

- Loaded dataset using Pandas
- Removed duplicates and handled missing values
- Cleaned and prepared data
- Performed exploratory data analysis
- Created visualizations


### 2. SQL Analysis

Performed business analysis using SQL queries:

- Customer segmentation
- Sales trends
- Category analysis
- Purchase behavior insights


### 3. Power BI Dashboard

Created an interactive dashboard showing:

- Sales performance
- Customer insights
- Product analysis
- Key business metrics



## Key Insights

- Identified customer purchasing patterns
- Analyzed sales trends
- Found important business metrics
- Created data-driven recommendations
